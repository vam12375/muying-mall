## 二、阿里云短信服务接入前提

### 2.1 阿里云控制台操作（需手动完成）

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 注册/登录阿里云 | https://www.aliyun.com |
| 2 | 完成实名认证 | 企业认证或个人认证均可 |
| 3 | 开通短信服务 | 控制台 → 短信服务 → 开通 |
| 4 | 创建 RAM 子账号 | 授予 `AliyunDysmsFullAccess` 权限 |
| 5 | 获取 AccessKey | 记录 AccessKey ID 和 AccessKey Secret |
| 6 | 申请短信签名 | 如 `母婴商城`，需审核 1-2 个工作日 |
| 7 | 申请短信模板 | 验证码模板，含变量 `${code}` |

### 2.2 短信模板示例

- **模板类型**：验证码
- **模板内容**：`您的验证码为${code}，5分钟内有效，请勿泄露给他人。`
- **模板编号**：审核通过后获得（如 `SMS_123456789`）

```java
    }
```

```java
     * @return 是否发送成功
     */

    /**
     * 验证短信验证码

     * @return 是否验证通过

```

**新建文件**：`src/main/java/com/muyingmall/service/impl/SmsServiceImpl.java`

核心逻辑：

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class SmsServiceImpl implements SmsService {

    private final com.aliyun.dysmsapi20170525.Client smsClient;
    private final AliyunSmsConfig smsConfig;
    private final StringRedisTemplate redisTemplate;

    @Override
    public boolean sendVerificationCode(String phone) {
        // 1. 检查发送间隔（60秒内不能重复发送）
        String intervalKey = CacheConstants.SMS_INTERVAL_KEY_PREFIX + phone;
        if (Boolean.TRUE.equals(redisTemplate.hasKey(intervalKey))) {
            throw new BusinessException(429, "发送过于频繁，请60秒后重试");
        }

        // 2. 检查每日发送次数限制
        String dailyKey = CacheConstants.SMS_DAILY_COUNT_KEY_PREFIX + phone;
        String countStr = redisTemplate.opsForValue().get(dailyKey);
        int count = (countStr != null) ? Integer.parseInt(countStr) : 0;
        if (count >= smsConfig.getDailyLimit()) {
            throw new BusinessException(429, "今日发送次数已达上限");
        }

        // 3. 生成随机验证码
        String code = generateCode(smsConfig.getCodeLength());

        // 4. 调用阿里云 SMS API 发送短信
        try {
            var request = new com.aliyun.dysmsapi20170525.models.SendSmsRequest()
                    .setPhoneNumbers(phone)
                    .setSignName(smsConfig.getSignName())
                    .setTemplateCode(smsConfig.getTemplateCode())
                    .setTemplateParam("{\"code\":\"" + code + "\"}");

            var response = smsClient.sendSms(request);

            if (!"OK".equals(response.getBody().getCode())) {
                log.error("短信发送失败: phone={}, code={}, message={}",
                        phone, response.getBody().getCode(), response.getBody().getMessage());
                throw new BusinessException("短信发送失败: " + response.getBody().getMessage());
            }
        } catch (BusinessException e) {
            throw e;
        } catch (Exception e) {
            log.error("短信发送异常: phone={}", phone, e);
            throw new BusinessException("短信发送失败，请稍后重试");
        }

        // 5. 存储验证码到 Redis
        String codeKey = CacheConstants.SMS_CODE_KEY_PREFIX + phone;
        redisTemplate.opsForValue().set(codeKey, code,
                Duration.ofMinutes(smsConfig.getCodeExpireMinutes()));

        // 6. 设置发送间隔
        redisTemplate.opsForValue().set(intervalKey, "1",
                Duration.ofSeconds(smsConfig.getSendIntervalSeconds()));

        // 7. 递增每日计数
        redisTemplate.opsForValue().increment(dailyKey);
        if (count == 0) {
            redisTemplate.expire(dailyKey, Duration.ofSeconds(CacheConstants.SMS_DAILY_EXPIRE_TIME));
        }

        log.info("短信验证码发送成功: phone={}", phone);
        return true;
    }

    @Override
    public boolean verifyCode(String phone, String code) {
        String codeKey = CacheConstants.SMS_CODE_KEY_PREFIX + phone;
        String storedCode = redisTemplate.opsForValue().get(codeKey);

        if (storedCode == null) {
            return false; // 验证码已过期
        }

        if (storedCode.equals(code)) {
            // 验证成功，立即删除验证码（一次性使用）
            redisTemplate.delete(codeKey);
            return true;
        }

        return false; // 验证码错误
    }

    /** 生成指定长度的数字验证码 */
    private String generateCode(int length) {
        var random = new java.security.SecureRandom();
        var sb = new StringBuilder();
        for (int i = 0; i < length; i++) {
            sb.append(random.nextInt(10));
        }
        return sb.toString();
    }
}
```

新增实现：

```java
@Override
public User findByPhone(String phone) {
    if (phone == null || phone.isEmpty()) {
        return null;
    }
    // 构建缓存键
    String cacheKey = CacheConstants.USER_KEY_PREFIX + "phone:" + phone;
    Object cacheResult = redisUtil.get(cacheKey);
    if (cacheResult != null) {
        return (User) cacheResult;
    }
    // 数据库查询
    User user = lambdaQuery().eq(User::getPhone, phone).one();
    if (user != null) {
        redisUtil.set(cacheKey, user, CacheConstants.USER_EXPIRE_TIME);
    }
    return user;
}

@Override
@Transactional(rollbackFor = Exception.class)
public User registerByPhone(String phone) {
    // 检查手机号是否已注册
    User existUser = findByPhone(phone);
    if (existUser != null) {
        throw new BusinessException("该手机号已注册");
    }

    User user = new User();
    user.setPhone(phone);
    // 自动生成用户名（手机号后4位 + 随机数）
    user.setUsername("user_" + phone.substring(7) + UUID.randomUUID().toString().substring(0, 4));
    user.setNickname("用户" + phone.substring(7));
    // 生成随机密码（用户可后续设置）
    user.setPassword(passwordEncoder.encode(UUID.randomUUID().toString()));
    user.setRole("user");
    user.setStatus(1);

    save(user);
    return user;
}
```

### 步骤 8：创建 SMS Controller 或扩展 UserController

**推荐方案**：在 `UserController` 中新增短信相关端点（与现有登录注册流程保持内聚）

**修改文件**：`src/main/java/com/muyingmall/controller/user/UserController.java`

新增注入和端点：

```java
// 新增注入
private final SmsService smsService;

/**
 * 发送短信验证码
 */
@PostMapping("/sms/send")
@Operation(summary = "发送短信验证码")
public Result<Void> sendSmsCode(@RequestBody @Valid SmsSendDTO smsSendDTO) {
    smsService.sendVerificationCode(smsSendDTO.getPhone());
    return Result.success(null, "验证码已发送");
}

/**
 * 手机验证码登录（自动注册）
 */
@PostMapping("/sms/login")
@Operation(summary = "手机验证码登录", description = "输入手机号和验证码登录，手机号未注册时自动创建账号")
public Result<LoginResponseDTO> smsLogin(@RequestBody @Valid SmsLoginDTO smsLoginDTO,
                                          HttpSession session) {
    // 1. 验证短信验证码
    if (!smsService.verifyCode(smsLoginDTO.getPhone(), smsLoginDTO.getCode())) {
        return Result.error(400, "验证码错误或已过期");
    }

    // 2. 查找或自动注册用户
    User user = userService.findByPhone(smsLoginDTO.getPhone());
    if (user == null) {
        // 手机号未注册，自动创建账号
        user = userService.registerByPhone(smsLoginDTO.getPhone());
    }

    // 3. 检查用户状态
    if (user.getStatus() != 1) {
        return Result.error(403, "账号已被禁用");
    }

    // 4. 生成JWT
    user.setPassword(null);
    String token = jwtUtils.generateToken(user.getUserId(), user.getUsername(), user.getRole());

    // 5. 缓存会话
    loginCacheService.cacheUserSession(token, user);

    // 6. 返回结果
    LoginResponseDTO responseDTO = new LoginResponseDTO(user, token);
    session.setAttribute("user", user);
    session.setMaxInactiveInterval(1800);

    return Result.success(responseDTO, "登录成功");
}
```

## 六、安全防护措施

| 防护项 | 实现方式 |
|--------|---------|
| 发送频率限制 | Redis Key: `sms:interval:{phone}`，TTL 60秒 |
| 每日发送限制 | Redis Key: `sms:daily:{phone}`，每日最多 10 次 |
| 验证码有效期 | Redis Key: `sms:code:{phone}`，TTL 5分钟 |
| 验证码一次性使用 | 验证成功后立即从 Redis 删除 |
| 手机号格式校验 | `@Pattern(regexp = "^1[3-9]\\d{9}$")` |
| IP 限流 | 可复用现有 `LoginRateLimiter`（可选增强） |
| 凭证安全 | AccessKey 通过环境变量注入，不硬编码 |

## 八、前端对接要点

### 8.1 一键登录/注册页面流程

```
1. 用户输入手机号
2. 点击"获取验证码"按钮 → POST /api/user/sms/send
3. 按钮进入60秒倒计时（前端控制）
4. 用户输入收到的验证码
5. 点击"登录/注册"按钮 → POST /api/user/sms/login
6. 成功后存储 JWT Token → 跳转到首页
```

### 8.2 前端需处理的状态码

| 状态码 | 含义 | 前端处理 |
|--------|------|---------|
| 200 | 成功 | 存储 Token，跳转 |
| 400 | 验证码错误/过期 | 提示重新获取 |
| 403 | 账号被禁用 | 提示联系客服 |
| 429 | 请求频率过高 | 提示等待 |
